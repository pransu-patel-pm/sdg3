# Village Health & Cleanliness Digital Record System

## Project Overview
This project is a web-based digital system developed as part of an NSS activity to record
anonymous village-level health and cleanliness data. The system helps identify patterns
and trends to support preventive health and sanitation planning.

## Objectives
- To record common health issues without collecting personal identities
- To track garbage and cleanliness problem locations
- To visualize data for preventive health awareness
- To support SDG-3 (Good Health & Well-Being)

## Features
- Anonymous health issue data entry
- Garbage and cleanliness issue recording
- Firebase Firestore database integration
- Simple dashboard showing total records
- Web-based system hosted on GitHub Pages

## Technologies Used
- HTML, CSS, JavaScript
- Firebase Firestore (Cloud Database)
- GitHub Pages (Hosting)

## Privacy & Ethics
- No personal names or contact details are collected
- The system does not diagnose diseases or prescribe medicines
- Data is used only for monitoring and preventive planning

## Deployment
The website is deployed using GitHub Pages and accessible publicly.
Firebase Firestore is used as the backend database.

## Disclaimer
This system is intended only for data recording and monitoring.
It does not replace medical professionals or healthcare services.
